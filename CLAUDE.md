# 扩散模型教程项目说明

## 项目目标
编写一份 扩散模型设计的由浅入深的教程HTML，要包含大量的习题和参考答案（答案默认折叠）。

## 工具说明
当需要时，可以通过 `gemini -p "深入回答：<要问的问题> -m gemini-2.5-pro"` 来获取 gemini-2.5-pro 的参考意见(gemini 系只问 gemini-2.5-pro 不问别人)
当需要时，可以通过 `echo "<要问的问题>"|llm -m 4.1 来获取 gpt-4.1 的参考意见
